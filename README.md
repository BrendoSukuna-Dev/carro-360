<h2>🖥 Projeto Concluido ✅ </h2> 

## 📋 Sobre o projeto
<p>Este projeto consiste em um visualizador interativo que permite ao usuário rotacionar um veículo em 360 graus utilizando o movimento do mouse.
  É uma solução ideal para e-commerces e catálogos digitais que buscam proporcionar uma visão detalhada de produtos.</p>


## ✦ Principais funcionalidades:

<p>➥ Rotação Interativa: O usuário pode clicar e arrastar para girar o carro, simulando um ambiente tridimensional.</p>
<p>➥ Loop Infinito: O sistema alterna entre 12 imagens de diferentes perspectivas, permitindo uma rotação contínua em ambas as direções.</p>
<p>➥ Lógica de Limiar (Threshold): Implementação de um deslocamento mínimo de 50px para acionar a troca de imagem, garantindo uma transição suave e controlada.</p>
<p>➥ Interface Imersiva: Fundo estilizado com gradiente radial e ocultação de barras de rolagem para foco total no produto.</p>
